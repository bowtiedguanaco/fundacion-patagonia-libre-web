# Checklist para CPE Santa Cruz

> Documento de trabajo curricular final para revisión interna, CPE Santa Cruz y aliados técnicos. No es acto administrativo, dictamen jurídico ni expediente definitivo. Toda denominación, carga horaria, alcance, práctica, requisito docente, infraestructura y trámite de reconocimiento queda sujeto a validación formal ante CPE Santa Cruz y, cuando corresponda, autoridad sanitaria/sectorial. No se promete validez nacional, matrícula, incumbencias reservadas ni habilitación automática.

## Objetivo de la reunión/consulta

Obtener guía formal o preliminar sobre el circuito para crear/incorporar un instituto terciario privado y presentar tecnicaturas superiores en Río Gallegos.

## Checklist institucional

- [ ] Confirmar autoridad/área competente para instituto superior privado nuevo.
- [ ] Confirmar documentación de titularidad institucional.
- [ ] Confirmar requisitos de sede, seguridad, habilitación edilicia e infraestructura.
- [ ] Confirmar régimen académico exigido.
- [ ] Confirmar documentación de autoridades, rectorado/dirección, secretaría académica y administración.
- [ ] Confirmar si se permite presentar proyecto institucional con carreras escalonadas.

## Checklist de plan de estudios

- [ ] Formato oficial del plan: campos, anexos, extensión, firmas.
- [ ] Denominación admitida de cada carrera.
- [ ] Título a otorgar admitido.
- [ ] Carga horaria mínima/máxima o rango esperado.
- [ ] Duración por años/cuatrimestres.
- [ ] Modalidad presencial y uso de virtualidad complementaria.
- [ ] Campos de formación requeridos.
- [ ] Régimen de correlatividades.
- [ ] Sistema de evaluación, promoción y acreditación.
- [ ] Perfil docente mínimo por espacio curricular.
- [ ] Formato de prácticas profesionalizantes.
- [ ] Requisitos de convenios y seguros.

## Preguntas por carrera

### Logística

- [ ] ¿La denominación “Logística y Abastecimiento Industrial” es aceptable?
- [ ] ¿Conviene incluir “Gestión” en denominación oficial o dejarla como marca comunicacional?
- [ ] ¿Qué nivel de práctica externa exige CPE para esta tecnicatura?
- [ ] ¿Qué perfiles docentes considera suficientes?

### Radiología

- [ ] ¿Qué área sanitaria debe intervenir antes/durante evaluación?
- [ ] ¿Qué convenios clínicos mínimos exige CPE?
- [ ] ¿Qué requisitos de radioprotección, seguros, bioseguridad y cupos deben documentarse?
- [ ] ¿Qué perfil debe tener la coordinación sanitaria?
- [ ] ¿Qué límites de alcance/incumbencia deben redactarse?
- [ ] ¿Cómo se articula matrícula/registro profesional si existiera? No prometer sin respuesta formal.

### Geomática

- [ ] ¿La denominación “Topografía, Geomática y Teledetección” es aceptable?
- [ ] ¿Qué límites exige CPE para no invadir agrimensura/ingeniería?
- [ ] ¿Qué equipamiento mínimo debe declararse?
- [ ] ¿Cómo documentar prácticas de campo, seguros y movilidad?
- [ ] ¿Drones pueden incluirse como herramienta formativa sin prometer habilitación externa?

## Preguntas críticas de aprobación

- [ ] ¿Conviene presentar primero una carrera o tres carreras juntas?
- [ ] ¿CPE recomienda apertura escalonada?
- [ ] ¿Qué plazos reales de evaluación maneja?
- [ ] ¿Qué observaciones suelen frenar expedientes?
- [ ] ¿Cómo se tramita reconocimiento/validez posterior si corresponde?
- [ ] ¿Qué no debe comunicarse públicamente antes de aprobación?

## Documentos a llevar

- [ ] Síntesis ejecutiva.
- [ ] Marco institucional.
- [ ] Plan curricular por carrera.
- [ ] Matriz comparativa.
- [ ] Borradores de convenios/cartas de intención.
- [ ] Inventario preliminar de infraestructura.
- [ ] CVs o perfiles tentativos de coordinadores/docentes si existieran.
