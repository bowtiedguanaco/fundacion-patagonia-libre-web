# Hoja de ruta del proyecto — FUN-4

- Fecha: 2026-04-30
- Proyecto: Instituto Terciario — Río Gallegos

## Objetivo

Convertir la hipótesis institucional en un proyecto aprobable, financiable y lanzable, con validación regulatoria y convenios mínimos antes de invertir en diseño curricular completo.

## Fase 0 — Decisión estratégica interna

Plazo: inmediato.

Entregables:

- Escenario elegido por Santiago.
- Titularidad institucional preliminar.
- Criterios de éxito: matrícula, posicionamiento, impacto político/institucional, sostenibilidad financiera.

Decisión recomendada:

- Adoptar Escenario B como hipótesis: Logística + Radiología + Geomática.

## Fase 1 — Validación regulatoria

Plazo: 0–15 días.

Acciones:

- Contactar CPE Santa Cruz.
- Confirmar requisitos para instituto privado nuevo.
- Confirmar si conviene presentar paquete de 3 carreras o avanzar por etapas.
- Ajustar nombres oficiales.
- Validar restricciones para salud/geomática.

Salida esperada:

- Checklist oficial o semi-oficial de requisitos.
- Ruta de expediente.
- Primer criterio de viabilidad por carrera.

## Fase 2 — Validación de aliados y prácticas

Plazo: 15–45 días.

Acciones:

- Logística: contactar empresas, cámaras, operadores, zonas francas, transporte, mineras/energéticas.
- Radiología: contactar Hospital Regional, clínicas y centros de diagnóstico.
- Geomática: contactar municipios, catastro, vialidad, obras públicas, ambiente, empresas de topografía/GIS/minería.

Salida esperada:

- 2–3 cartas de intención por eje o al menos reuniones positivas documentadas.
- Mapa de prácticas posibles.
- Lista de docentes/contactos técnicos potenciales.

## Fase 3 — Prefactibilidad económica

Plazo: 30–60 días.

Acciones:

- Estimar matrícula esperada por carrera.
- Definir arancel objetivo.
- Calcular docentes/hora/costo.
- Identificar equipamiento mínimo.
- Estimar break-even.
- Evaluar si conviene lanzar 1, 2 o 3 carreras en primera cohorte.

Salida esperada:

- Matriz financiera simple por carrera.
- Decisión de cohorte inicial.

## Fase 4 — Diseño académico institucional

Plazo: 45–90 días.

Acciones:

- Redactar proyecto institucional.
- Diseñar perfiles de egreso.
- Diseñar mallas preliminares.
- Diseñar régimen de prácticas profesionalizantes.
- Identificar directores/coordinadores de carrera.
- Preparar documentación de infraestructura.

Salida esperada:

- Carpeta académica preliminar lista para revisión externa.

## Fase 5 — Presentación formal y lanzamiento público

Plazo: depende de CPE.

Acciones:

- Presentar expediente.
- Responder observaciones.
- Preparar marca, landing, campaña de interés y preventa si legalmente viable.
- Activar alianzas y comunicación institucional.

Salida esperada:

- Aprobación o ruta concreta de aprobación.
- Plan de lanzamiento comercial/institucional.

## Semáforo actual

- Visión institucional: verde.
- Selección de carreras: verde para hipótesis, amarillo hasta validación externa.
- Regulación: amarillo.
- Convenios: rojo/pendiente.
- Finanzas: rojo/pendiente.
- Carpeta académica: amarillo inicial.
